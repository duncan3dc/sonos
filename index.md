A PHP library for interacting with [Sonos](https://www.sonos.com/) speakers.

It allows you to manage what's playing now, which speakers are grouped together, playlists, alarms, and hopefully anything you would want to.
<br><br>

<p class="message-api">Full <a href='{{ site.baseurl }}/api/packages/Application.html'>API documentation</a> is also available.</p>

If you're upgrading to v3 from v2 then there's some [notes here](v3/) and the [CHANGELOG](https://github.com/duncan3dc/sonos/blob/master/CHANGELOG.md) lists some things to consider.
